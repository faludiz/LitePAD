## A Simple Text Editor

- No tabs: single file per window
- No AI functions
- No syntax highlight
- No automatic updates
- No online functions
- No warranty

## Supported functions

- File: New, Open, Save, Save As
- Encoding: Auto detect for open and save
- Edit: Cut, Copy, Paste, Find, Replace, Jump, Set font, Word wrap
- Insert: Date, Time, Date and Time, GUID, User
- Auto save and load: Window position, Font
- You can translate to Your language (thx)
- Dark mode: if set

## Translations

- [Fork](https://github.com/faludiz/LitePAD/fork)
- Use [PoEdit](https://poedit.net/) or Your favourite language file editor to create the Your language file from `languages\litepad.pot`.
- Save language file to `litepad.{languagecode}.po` where `{languagecode}` is Your language code, i.e.: `fr`, `de`, etc.
- If You want to test/use, You must copy to the `bin\languages` folder.
- After everything is fine, make a [Pull request](https://github.com/faludiz/LitePAD/compare)

## Build

This program is written in [Lazarus](https://www.lazarus-ide.org/). The only dependency is the [MetaDarkStyle](https://github.com/zamtmn/metadarkstyle) package, which You can install with the Lazarus's Online Package Manager.
